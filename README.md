# Sessions

A React and Vite based sessions calendar interface for managing psychologist sessions, schedules, session history, and session details.

## Tech Stack

- React
- Vite
- Carbon React
- FullCalendar
- Sass

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run lint checks:

```bash
npm run lint
```

## Main Features

- Session calendar with month, week, day, and schedules views
- Mini calendar navigation and filters
- Session scheduling drawer
- Session history table
- Event details popover
- Video call session screen
